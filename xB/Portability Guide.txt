Use the UTF-8 encoding, although the official specification mandates UTF-16.

Do not use final commas in object and array literals:
{a: 123,};                 // Avoid.
{a: 123};

Do not access string characters as array elements:
myString[2];               // Avoid.
myString.charAt(2);

Do not use the start token of block comments inside them:
/* /* */                   // Avoid.
/*    */

Do not check if a value is a function with the "instanceof" keyword:
f instanceof Function      // Avoid.
typeof f == 'function'

Do not create anonymous functions from strings with "eval":
eval('function() {}');     // Avoid.
function() {}
